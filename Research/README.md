# Research

> [INFO] tagged content — tools, training labs, articles, and anything interesting that isn't active threat intel.

## Subfolders
- **Tools/** — security tools, platforms, scanners, detection frameworks
- **Articles/** — research papers, blog posts, technique deep-dives
- **Training/** — labs, courses, exercises, learning resources

## vs Threat-Hunting
If it came in tagged `[INTEL]` and describes an active threat, campaign, or detection — it goes in `Threat-Hunting/` or `Detection-KQL/`.
If it came in tagged `[INFO]` and is a tool, resource, or interesting read — it goes here.
