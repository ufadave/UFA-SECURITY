# INFO -- TrustedSec: Coverage-Driven Sustained Testing (CDST) with Agentic Workflows

**Source:** https://trustedsec.com/blog/coverage-driven-sustained-testing-cdst-agentic-workflows
**Date:** 2026-05-20
**Author:** TrustedSec

---

## What It Is

TrustedSec's model for applying agentic AI workflows to continuous security testing -- specifically a graph-oriented approach where the attack surface is modelled as a node graph and autonomous agents navigate it to identify coverage gaps. CDST frames security testing as a sustained programme rather than point-in-time engagements, using agentic tools to continuously retest as the environment changes. The graph model allows the testing programme to track which paths have been tested, which have not, and which have changed since the last test cycle -- providing measurable coverage metrics analogous to code coverage in software testing.
